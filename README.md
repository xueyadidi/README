## **图片**
****
基本格式：
```
![alt](URL title)
```
- alt为图片加载失败时的替换文本
- title表示鼠标悬停在图片时的显示文本，title需要加引号

|#|语法|效果|
|----|----|-----|
|1|`![baidu](http://www.baidu.com/img/bdlogo.gif "百度logo")`|![baidu](http://www.baidu.com/img/bdlogo.gif "百度logo")|
|2|`![foryou][foryou]`|![foryou][foryou]|

例2使用了URL标识符的形式，在链接一节有介绍。
